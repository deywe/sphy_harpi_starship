# SPHY Core — Parquet Validator & Visualizer

This module is the official auditing and visual reproduction tool for the **Harpia Quantum Deeptech** ecosystem. It is engineered to process massive datasets in `.parquet` format, ensuring that every frame of the "Propulsion by Geometric Tensioning" simulation is authentic through rigorous **SHA-256** hash verification.

## 🔍 Overview: What does this tool do?

This script serves as the **Data Auditing Layer** for the HarpiaOS environment. While the real-time viewer focuses on aesthetics and fluid motion, this validator ensures the **mathematical stability and integrity** of the datasets generated by the SPHY Core.

### 1. Cryptographic Auditing (SHA-256)

The script reconstructs the state of each frame from the `.parquet` columns. It generates a canonical JSON payload and recalculates the **SHA-256** hash.

* **Entropy Detection:** If a single bit is altered in the dataset—whether by disk write errors or unauthorized tampering—the validator identifies the exact corrupted frame.
* **Determinism:** It guarantees that the simulation you see today is bit-for-bit identical to the one originally processed.

### 2. Validation Report Anatomy (`sphy_report.png`)

The validator generates a technical dashboard divided into 8 analytical modules:

* **XY Orbital Map:** A long-exposure view of all trajectories, confirming if orbits adhere to the Golden Ratio ($\phi$).
* **SHA-256 Status Grid:** A point matrix where **Green** represents integrity and **Red** represents data corruption.
* **Z-Variation (Inclination):** Monitors the vertical stability of celestial bodies, essential for validating geometric tensioning propulsion.
* **Angular Velocity:** Validates that planetary kinematics respect Keplerian laws integrated into SPHY.
* **Starship XZ Path:** Focuses on altitude and the zero-mass transition of the ship during transit.
* **Hash Fingerprint:** A visual comparison of the first and last frame hashes, showing bit variation over time.
* **Integrity Stats:** An executive summary including line counts, frames, and final validation status.
